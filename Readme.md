# Greenhouse System — SA-MP 0.3.7

Система теплиц для SA-MP сервера.

## Архитектура

- **Хранилище**: SQLite (встроенный a_sqlite)
- **Динамические объекты**: Streamer Plugin v2.9.6
- **Таймеры**: SetTimerEx на каждую активную теплицу
- **Загрузка/выгрузка**: OnPlayerConnect / OnPlayerDisconnect

## Команды

| Команда | Описание |
|---|---|
| `/creategh` | Создать теплицу на текущей позиции |
| `/upgradegh` | Улучшить теплицу рядом (x2 скорость) |
| `/collectgh` | Собрать урожай (только при 100%) |
| `/ghstatus` | Статус всех теплиц |

## Логика роста

- Базовый: 1% каждые 6 секунд = 100% за 10 минут
- Улучшенный: 1% каждые 3 секунды = 100% за 5 минут
- Визуал: 3 стадии (0–33%, 34–66%, 67–100%) через динамические объекты

## Оптимизация (1000 онлайн)

- Таймер создаётся на объект, не на игрока
- Таймер убивается при дисконнекте, пересоздаётся при коннекте
- SQL-запросы только при коннекте/дисконнекте и командах
- Объекты streamer создаются/удаляются только при смене стадии

## Установка

1. Положить `streamer.dll` в `/plugins`
2. Добавить `plugins streamer` в `server.cfg`
3. Скомпилировать `gamemodes/greenhouse.pwn`
4. Прописать `gamemode0 greenhouse` в `server.cfg`
